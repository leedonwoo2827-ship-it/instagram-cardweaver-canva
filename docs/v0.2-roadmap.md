# CardWeaver v0.2 로드맵

v0.1 범위는 Instagram 단독. v0.2 이후 확장 항목을 기록한다.

## 트랙 A: 채널 확장

각 채널 확장은 [knowledge/channel-style-research.md](../knowledge/channel-style-research.md) 의 해당 섹션을 시드로 사용. 그 문서에 18개 한국 채널의 비율·글풍·훅/CTA·해시태그·발행리듬 초안 + 🔎 조사 필요 항목이 이미 정리되어 있음. 스프린트 시작 시: 해당 섹션 → `channels/<name>/RESEARCH.md` 로 복사 → 🔎 플래그 조사로 채움 → `profile.json` + `templates/` 구현.

우선순위 순:

1. **Threads** (Meta, 최근 한국 MAU 급성장)
   - Instagram 과 Meta 공유 → 프로필·태그 재사용 용이
   - 1:1 비율, 3~5장으로 더 타이트
   - `channels/threads/profile.json` 작성 + `templates/*.html` Instagram 에서 비율만 조정
   - 예상 작업량: 1~2일

2. **LinkedIn** — 🔎 사양 조사 선행 필요
   - Document Post 캐러셀 vs 일반 이미지 포스트 선택 조사
   - B2B 톤: 수치·근거 강조
   - 교육사업 실적 시리즈와 시너지 큼
   - `channels/linkedin/RESEARCH.md` → 결과 반영 후 구현

3. **Facebook** — 🔎 사양 조사 선행 필요
   - 2026년 알고리즘 변화, 링크 미리보기 우선순위
   - 권장 비율 (1:1 vs 4:5) 재확인
   - 카드뉴스 도달력이 실제로 있는지 재평가

4. **네이버 브런치** — 🔎 사양 조사 선행 필요
   - 커버 이미지(1200×630) + 본문 삽입 이미지(1200×800) 이원화
   - SEO 헤딩 자동 삽입 규칙
   - 책 마케팅에 유리 (장기 SEO)

각 채널은 `channels/<name>/RESEARCH.md` 로 조사 결과를 먼저 정리한 뒤 `profile.json` + `templates/` 구축.

## 트랙 B: 렌더 도구 확장

### B1. Mirra MCP 어댑터 검증

- `/card-render --target mirra` 옵션 추가
- Canva 와 A/B 테스트 (평가 지표는 `docs/TOOL-COMPARISON.md` 참조)
- Mirra 승리 시 기본 렌더 전환

### B2. Figma Auto Layout Export

- `/card-render --target figma` — Figma REST API 용 JSON 생성
- 디자이너 수작업 최종 조정 단계 지원

### B3. MCP 이미지 생성 백엔드

- FlowGenie(기본) 외에 **Nano Banana** 또는 **Pixa** MCP 선택지 제공
- 옵션 `--image-backend flowgenie|nano-banana|pixa`
- 종량제 비용 경고 UI

## 트랙 C: 파이프라인 연동

### C1. SceneWeaver 연동 (카드뉴스 → 숏폼 영상)

- CardWeaver 의 `instagram/preview/*.png` + TTS 음성 → SceneWeaver 로 자동 전달
- 9:16 릴스/쇼츠 출력
- 스크립트 공유: `card_script.json.cards[].body` → 음성 나레이션

### C2. ScriptForge 입력 공유

- 책 소스는 이미 ScriptForge 에서 가공한 `ch{NN}_script.json` 이 있을 경우, CardWeaver 가 그 파일을 직접 입력으로 수용
- 새로 분해하지 않고 scenes[] 에서 핵심만 추려 카드로 변환

## 트랙 D: 구조 정리

### D1. 독립 패키지 분할

- `cardforge` (콘텐츠→스크립트) 와 `cardrender` (스크립트→출력) 를 별도 npm 패키지로 분리
- 재사용·테스트 용이
- 새 폴더: `260424-cardforge/`, `260425-cardrender/`

- `claude-plugin` 마켓플레이스 등록 (SceneWeaver 와 같은 owner)
