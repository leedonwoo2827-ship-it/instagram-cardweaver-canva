# 2025 K-공기업 임직원 교육 플랫폼 구축 사업

## 과제 개요

**고객**: K-공기업 (임직원 약 3,500명)
**기간**: 2025년 3월 ~ 2025년 12월 (10개월)
**규모**: 총 사업비 비공개, 학습자 3,500명 전원 커버

## 배경

기존 LMS 는 2018년 도입 이후 7년간 큰 업데이트 없이 운영되어, 모바일 접근성·콘텐츠 관리·학습 분석 세 측면에서 한계가 명확했다. 2024년 말 감사원 지적 이후 전면 재구축이 결정되었다.

## 수행 실적

- 기존 LMS 콘텐츠 1,247개 마이그레이션 (손실률 0.3%)
- 신규 콘텐츠 200개 제작 (마이크로러닝 5분 포맷 중심)
- 플랫폼 월 활성 사용자 92% (도입 전 41% → 도입 후 92%)
- 학습 이수율 평균 34%p 개선
- NPS 점수 +62점

## 접근 방식

1. **Discovery**: 6주 현장 인터뷰 48건 + 기존 플랫폼 사용 데이터 분석
2. **Design**: 사용자 저니맵 기반 신 플랫폼 설계, 모바일-퍼스트
3. **Build**: 4개월 애자일 개발, 2주 단위 현업 검증
4. **Launch**: 파일럿 3개 부서 2개월 → 전사 확산

## 핵심 차별점

- **마이크로러닝 전환**: 기존 60분 단위 이러닝을 5분 단위 240개로 재편
- **학습 분석 대시보드**: 부서장용 주간 리포트 자동 발송
- **모바일 네이티브**: iOS/Android 네이티브 앱, 웹은 보조

## 성과

- 2025년 12월 기준 이수율 34%p 개선
- 만족도 조사 4.6/5.0 (도입 전 3.1/5.0)
- 유사 공기업 3곳에서 벤치마킹 문의
